## Log in to GenSX Cloud (optional)

If you haven't already, download the GenSX CLI and log in to GenSX Cloud using the following commands:

```bash
# Install the CLI
npm i -g gensx

# Log in to GenSX Cloud
gensx login
```

You’ll be redirected to the GenSX website and will need to create an account if you don’t have one already.

## Download the template

To download the template, run the following command:

<TabSection tab="chat-ux">
```bash
gensx template clone chat-ux
```

</TabSection>

<TabSection tab="deep-research">
```bash
gensx template clone deep-research
```

</TabSection>

<TabSection tab="draft-pad">
```bash
gensx template clone draft-pad
```

</TabSection>

<TabSection tab="client-side-tools">
```bash
gensx template clone client-side-tools
```

</TabSection>

This will create a new directory with the template code.

## Run the template

Now that you've cloned the template, navigate to the template directory:

<TabSection tab="chat-ux">
```bash
# Navigate to the template directory
cd chat-ux

# Export environment variables
export TAVILY_API_KEY=your-api-key
export ANTHROPIC_API_KEY=your-api-key

# Start the next.js app and gensx dev server
pnpm run dev
```
</TabSection>

<TabSection tab="deep-research">
